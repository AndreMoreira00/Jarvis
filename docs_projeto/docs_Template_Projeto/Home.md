# Home — _Template_Projeto

> **Este é um template.** Para iniciar um novo projeto:
> 1. Duplique a pasta `docs_Template_Projeto/` e renomeie para o nome do projeto (ex.: `Produto_X/`).
> 2. Ajuste o frontmatter de todas as notas trocando `project: _Template_Projeto` pelo nome real.
> 3. Abra `00_SGI_Aplicado/Procedimentos_Aplicaveis` e liste as normas/procedimentos que se aplicam ao projeto.
> 4. Preencha o cronograma em `01_Gestao/Cronograma/` e a primeira ADR em `01_Gestao/Decisoes_Tecnicas/`.
>
> Consulte [[../HOWTO|HOWTO]] e [[../CONVENCOES|Convencoes]] na raiz de `docs_projeto/`.
